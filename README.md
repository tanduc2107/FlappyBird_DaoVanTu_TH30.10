## Giới thiệu Đề tài
Dự án phát triển trò chơi Flappy Bird 2D cổ điển bằng ngôn ngữ lập trình Python. Giao diện trò chơi được tối ưu hóa bằng hệ thống Vector đồ họa (Canvas) của thư viện tiêu chuẩn `Tkinter`, giúp mã nguồn đóng gói gọn nhẹ, mượt mà ở tốc độ 60 FPS và chạy độc lập trên mọi cấu hình máy tính mà không cần cài đặt thêm thư viện mở rộng từ bên ngoài.

## Hướng dẫn cài đặt và vận hành
1. Yêu cầu máy tính đã cài đặt sẵn Python 3.x.
2. Tải file `main.py` về máy.
3. Chạy file `main.py` bằng các công cụ lập trình (IDE) như Spyder, PyCharm hoặc VS Code.

## Phím điều khiển
- Ấn phím **Space (Phím cách)** để điều khiển chú chim vỗ cánh bay lên vượt chướng ngại vật.
- Ấn phím **Space** khi kết thúc lượt để chơi lại màn mới ngay lập tức.
